~/about

I'm Denis Krüger — Information Security student at THWS Würzburg, focused on offensive security and understanding how attack chains are constructed end-to-end.

I approach security through structured reconnaissance, deep enumeration, and controlled exploitation in lab environments. Currently working through TryHackMe and HackTheBox alongside my studies, building practical skills in web application testing, privilege escalation, and attack documentation.

I value understanding why an exploit works — not just how to reproduce it.

pwgen-lite

python

Minimal CLI password generator built around secure randomness and cryptographic correctness. Implements secrets-based generation, HMAC-SHA256 deterministic mode with rejection sampling, entropy estimation, and brute-force time modelling. Full pytest suite included.

mfind

c

Simplified reimplementation of the Unix find utility in C. Recursive directory traversal, name/type/size/depth filtering, and optional parallel traversal across start directories using POSIX threads. Verified memory-safe with Valgrind (zero leaks).
