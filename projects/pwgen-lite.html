--- layout: page title: pwgen-lite subtitle: "Minimal CLI password generator built around cryptographic correctness." ---

pwgen-lite is a command-line password generator written in Python. The focus is on doing one thing correctly: generating passwords you can actually trust, with honest entropy estimates and no magic.

Random mode uses Python's secrets module (CSPRNG). Deterministic mode derives passwords from a master key and label using HMAC-SHA256 with rejection sampling so the output distribution is exactly uniform over the requested character set.

~/features

  • >

    Secure randomness

    secrets-based generation — no random module.

  • >

    Deterministic mode

    HMAC-SHA256 + rejection sampling. Same key+label always yields the same password.

  • >

    Entropy estimation

    Reports bits of entropy and estimated brute-force time at various attack rates.

  • >

    Full test suite

    pytest suite covering generation, determinism, entropy calculations, and edge cases.

~/usage 

python pwgen.py --length 24 --charset alnum
python pwgen.py --length 32 --deterministic --key "masterkey" --label "github"
